Description

Summary

글 작성 및 삭제, 그리고 검색이 가능한 게시판

About Project

게시글 작성
- 작성자 닉네임, 게시글 제목과 본문을 입력 후 제출 버튼을 누릅니다. 
- 중요한 게시글이라면 중요도 체크박스 버튼을 선택할 수 있습니다. 
게시글 삭제
- 삭제 버튼을 누릅니다. 
게시글 검색
- 작성자 닉네임과 제목으로 검색이 가능합니다. 

How to Use

톰켓 9버전에서 동작가능합니다. war 패키징하여 사용해보세요.
